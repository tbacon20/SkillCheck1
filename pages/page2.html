How can we help you?


Step 1: Call cloud API

API Gateway request

The user sends a POST request to API Gateway through a jquery form. This API request will trigger a lambda function.

Step 2: Parse Request

AWS Lambda function

When the lambda function is triggered, it will take the HTTP request and send a new record to a DynamoDB table with the JSON payload information from the form.

Step 3: Store message

Amazon DynamoDB Table

All items submitted to the form will be stored in a table where they can be viewed individually or seen as a table with email, phone number, name, and the email message.
